#### 支持任务类型

1. 显示下载(Explicit)

2. 隐式下载(Silence)

     1. 没有进度提醒
     2. 下载成功，返回成功的结果
     3. 下载过程不关联通知栏，消息抽屉
     4. 提供下载的重试。（默认3次重试）
     5. 最终下载失败返回错误结果
